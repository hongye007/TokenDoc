## 注册与登录

@TD_BRAND_DISPLAY@ 支持**账号密码注册**与多种**第三方 OAuth 一键登录**。建议按 **先注册、后登录** 的顺序完成首次使用。

### 注册流程

1. 打开 [@TD_BRAND_DISPLAY@ 官网](@TD_MAIN_SITE@)，点击右上角「**注册**」，或直接访问 [注册页](@TD_REGISTER_URL@)。

![注册页](/quickstart/register.png)

2. 填写用户名与密码（遵循页面提示的长度与格式要求）。
3. 填写邮箱，点击「**获取验证码**」，将收到的验证码填入输入框。
4. 点击「**注册**」完成账号创建；成功后通常会跳转至首页或控制台。

### 账号密码登录

1. 在官网点击右上角「**登录**」，或直接访问 [登录页](@TD_LOGIN_URL@)。

![登录页](/quickstart/login.png)

2. 在登录页输入**用户名或邮箱**与**密码**，点击「**继续**」完成登录。

### 第三方 OAuth 登录

若使用 GitHub、Discord、LinuxDO 等第三方账号，点击登录页下方对应平台的图标，跳转至授权页面完成授权后会自动登录。

### 忘记密码

在登录页点击「**忘记密码？**」，输入注册邮箱；系统将发送重置链接，打开邮件中的链接即可设置新密码，原密码随即失效。
