## 令牌管理

**令牌**即调用开放接口时使用的 API Key：可为不同场景分别创建，单独设置配额、可用模型、IP 与分组等。进入方式：侧栏 **令牌管理**，或浏览器打开 [控制台 · 令牌管理](@TD_CONSOLE@/token)。

![令牌管理列表页：侧边栏与添加令牌](/settings/token/token-list.png)

列表中会展示名称、状态、额度、分组、密钥（默认脱敏）、可用模型、IP 限制、创建与最近使用时间等；支持关键字与密钥筛选，以及批量复制、删除所选令牌。

### [创建令牌](#创建令牌)

1. 点击 **「添加令牌」**，打开创建弹窗。
2. 填写 **名称**（建议按环境或用途命名，例如「生产网关」「本地调试」）。
3. 按需选择 **令牌分组**；不选时一般沿用账户默认分组。
4. 设置 **过期时间**，或使用「永不过期 / 一个月 / 一天 / 一小时」等快捷项。
5. **新建数量** 大于 1 时，系统会在名称后追加随机后缀，便于批量创建互不冲突的密钥。

![创建新的令牌弹窗](/settings/token/token-create-modal.png)

填写完成后点击 **「提交」** 保存；若开启无限额度等选项，实际消耗仍受 **账户剩余额度** 约束。

### [高级选项说明](#高级选项说明)

弹窗中常见字段含义如下（以控制台实际展示为准）：

| 配置项 | 说明 |
| --- | --- |
| 过期时间 | 到达后即不可用；支持快捷选择与自定义时间。 |
| 剩余配额 | 该令牌允许消耗的上限；用尽后令牌失效。 |
| 无限额度 | 关闭后按上面额度限制；开启后令牌侧不封顶，但仍不能超过账户总可用额度。 |
| 模型限制 | 仅允许列出的模型；留空通常表示不额外限制（仍受渠道与账户策略影响）。 |
| IP 白名单 | 仅允许来自这些 IP 的请求；留空表示不按 IP 限制。 |
| 分组 | 指定令牌走哪一类渠道分组，影响路由与计费策略。 |

### [查看密钥与接入客户端](#查看密钥与接入客户端)

创建成功后会回到列表。在 **密钥** 列可点击 **眼睛图标** 临时查看完整 Key，并配合复制操作保存到安全位置（密码管理器或私密配置，勿写入公开仓库）。

行内 **「聊天」** 下拉中会列出已适配的第三方客户端（如 CC Switch、Cherry Studio 等），可按需一键带入当前令牌，减少手工填地址与密钥的步骤。

![列表中查看密钥与聊天应用集成](/settings/token/token-table-actions.png)

**安全提示：** 令牌与账户资金、调用额度直接相关，勿通过聊天、邮件或截图外泄；泄露后请尽快在控制台 **禁用 / 删除** 并新建令牌。

### [编辑与删除](#编辑与删除)

- **编辑**：在目标行点击 **「编辑」**，可调整名称、额度、模型与 IP 等配置；**密钥字符串本身通常不可在编辑页中修改**，需新建令牌以轮换密钥。
- **删除**：点击 **「删除」** 并经确认后，该令牌立即作废且不可恢复；请确认依赖方已更新为新 Key。
