## 模型广场

浏览全站可用模型、筛选供应商与计费类型，并查看价格与复制模型名称。

### 访问模型广场

点击顶栏「**模型广场**」进入（具体入口以控制台为准）。

![模型广场：筛选侧栏、搜索与模型卡片列表](/settings/pricing/model-plaza.png)

### 浏览与筛选

左侧可按供应商、可用令牌分组、计费类型、标签、端点类型等筛选；主区域为模型卡片，可切换充值价格显示、币种、表格视图等。

### 搜索模型

在顶部搜索框输入关键词，可模糊匹配模型名称，快速定位目标模型。

### 价格说明

#### 计费规则

*   **输入价格**：每 1K 输入 Token 消耗的配额
*   **输出价格**：每 1K 输出 Token 消耗的配额
*   实际消耗 = Token 数量 ÷ 1000 × 对应单价

#### 分组差异

不同分组的用户可能享有不同的计费倍率，具体以实际扣减为准，可在充值页查看当前余额变化。
