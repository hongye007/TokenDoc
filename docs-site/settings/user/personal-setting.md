## 个人设置

在 AtomFlow（原流）控制台管理账号信息、安全选项与第三方绑定。可点击右上角头像选择「**个人设置**」，或进入 **个人中心 → 个人设置**；也可直接访问 `/console/personal`。

### 进入个人设置

![从控制台进入个人设置](/settings/personal/entry-from-dashboard.png)

个人设置页分为 **账户管理**、**偏好设置**、**其他设置** 等板块，布局如下。

![个人设置页总览：账户管理、偏好设置与其他设置](/settings/personal/personal-overview.png)

### 账户管理

「**账户管理**」包含 **账户绑定** 与 **安全设置** 两个标签：前者用于邮箱、第三方账号与用户名等；后者包含系统访问令牌、密码、Passkey、两步验证与删除账户等。

#### 账号绑定与系统访问令牌

- **用户名**：在相应输入框修改后点击「保存」。
- **绑定邮箱**：在 **账户绑定** 标签下填写邮箱，点击「发送验证码」，输入验证码完成绑定。
- **绑定第三方账号**：在同一标签中选择 GitHub、Discord 等平台，点击「绑定」，在授权页完成流程后返回控制台确认「已绑定」（具体以页面展示为准）。

在 **安全设置** 标签下可查看 **系统访问令牌**（用于 API 调用的身份凭证）。请妥善保管；若怀疑泄露，可点击「重新生成」。**重新生成后旧令牌立即失效**，依赖旧令牌的集成需同步更新。

![账户管理：安全设置中的系统访问令牌等](/settings/personal/personal-account-token.png)

在 **可用模型**（或同类入口）中可查看当前账号可调用的模型列表，支持搜索与复制模型名称。

#### 密码管理

在 **安全设置 → 密码管理** 中点击「**修改密码**」，在弹窗中依次填写原密码、新密码与确认新密码，点击「确定」保存。

![修改密码弹窗](/settings/personal/personal-change-password.png)

#### Passkey 登录

1. 在 **安全设置** 中找到「**Passkey 登录**」。
2. 点击「**注册 Passkey**」，按浏览器或系统提示完成验证。
3. 完成后即可在支持的场景下使用 Passkey 登录。

#### 两步验证设置

开启两步验证后，登录时除密码外还需输入验证器 App 中的动态码，降低被盗风险。

1. 在手机上安装验证器 App（推荐 Google Authenticator 或 Microsoft Authenticator）。
2. 在 **安全设置** 中找到「**两步验证**」，点击「**启用验证**」。

![两步验证：在安全设置中点击启用验证](/settings/personal/2fa-section.png)

3. 用验证器扫描页面上的二维码；若无法扫描，可手动输入密钥。

![两步验证：扫描二维码](/settings/personal/2fa-qrcode-modal.png)

4. 将验证器中的 6 位动态码填入页面，完成验证。
5. 按提示保存**备用码**（建议截图或离线保存）；备用码通常只展示一次，丢失且无法使用验证器时需联系管理员处理。

#### 删除账户

在 **安全设置** 底部的 **删除账户** 为不可逆操作，请谨慎确认后再执行。

### 偏好设置

「**偏好设置**」用于界面语言等个人偏好；设置会同步到已登录设备，并可能影响 API 返回错误信息的语言。

#### 语言偏好

在 **语言偏好** 中选择界面语言（如简体中文），选择后一般会自动保存。

![偏好设置：语言偏好](/settings/personal/personal-language-preference.png)

### 其他设置

「**其他设置**」涵盖通知、边栏等子项，通过顶部标签切换。

#### 邮件通知

在 **通知配置** 标签中，将通知方式选为「**邮件通知**」，按需填写**额度预警阈值**与**通知邮箱**（留空可使用账号绑定邮箱），点击「**保存设置**」生效。

![其他设置：通知配置（邮件通知）](/settings/personal/notification-mail.png)
