## 使用记录

查看每次 API 调用及相关操作的记录，支持按时间、模型、令牌、日志类型等条件筛选。左侧导航进入「**使用日志**」，或直接访问 `/console/log`。普通用户只能看到自己的记录。

![使用日志：列表与统计摘要](/settings/log/log-list.png)

列表每行展示一条记录，包含时间、令牌、分组、类型、模型、输入输出 Token、花费、IP、详情等（具体列可通过「列设置」调整）。

### [设置过滤条件](#设置过滤条件)

1. 在页面顶部选择**时间范围**，按需填写**令牌名称**、**模型名称**、**分组**、**Request ID** 等条件。
2. 使用**日志类型**下拉框限定范围（如全部、充值、消费、管理、系统、错误等）。
3. 点击「**查询**」应用条件；「**重置**」可清空筛选。

![使用日志：筛选栏与日志类型](/settings/log/log-filters.png)

### [查看过滤结果](#查看过滤结果)

设置完成后点击「查询」，表格会刷新为当前条件下的记录；可结合顶部的消耗额度、RPM、TPM 等摘要查看概况。

![使用日志：查询后的记录示例](/settings/log/log-filtered.png)

### [访问数据看板](#访问数据看板)

左侧导航点击「**数据看板**」，或直接访问 `/console`，可查看账户余额、使用统计、资源消耗、性能指标以及模型数据分析等图表。

![数据看板：账户与模型数据总览](/settings/log/dashboard-overview.png)

### [查看统计图表](#查看统计图表)

数据看板中以折线图、柱状图等展示调用与消耗趋势；切换不同分析标签可查看消耗分布、调用趋势等。将鼠标悬停在图表上可查看更细粒度数据。
