## 钱包管理

配额是平台内部计费单位，消耗量与模型倍率、调用量等相关。左侧导航进入「**钱包管理**」，或直接访问 [控制台 · 钱包管理](@TD_CONSOLE@/topup)。

![钱包管理：账户充值与邀请奖励](/settings/topup/wallet-overview.png)

页面通常分为 **账户充值**（在线支付、额度选择、兑换码）与 **邀请奖励**（邀请链接、收益划转）等板块。支持的方式以控制台实际展示为准（例如支付宝、Stripe 等）。

### [选择充值金额并完成支付](#选择充值金额并完成支付)

1. 在 **账户充值** 中查看当前余额、历史消耗等统计。
2. 填写或选择 **充值数量**，并选择 **支付方式**（如支付宝、Stripe）。
3. 可在 **选择充值额度** 中点击预设档位快速填入。
4. 确认 **实付金额** 后按页面提示发起支付，在支付渠道完成付款；返回后余额会更新。需要核对流水时，可点击「**账单**」查看。

### [输入兑换码](#输入兑换码)

1. 在同一页的 **兑换码充值** 区域找到输入框。
2. 粘贴或输入管理员提供的兑换码。

![兑换码充值区域](/settings/topup/wallet-redeem.png)

### [完成兑换](#完成兑换)

1. 点击「**兑换额度**」按钮，系统校验兑换码。
2. 兑换成功后按提示确认到账，余额同步更新。

### [邀请奖励](#邀请奖励)

每个账号有专属邀请链接。被邀请用户注册并产生充值等行为后，你可按平台规则获得奖励；在 **邀请奖励** 中可查看待使用收益、总收益与邀请人数，将奖励 **划转到余额** 后即可用于消费。可复制邀请链接分享给好友。

![邀请奖励：收益统计与邀请链接](/settings/topup/wallet-invite.png)

奖励细则与比例以页面「奖励说明」及运营政策为准。
